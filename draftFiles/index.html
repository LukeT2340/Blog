SwiftUI Tutorial: Creating a responsive IOS Login Screen

In SwiftUI, you can easily create a login screen for your app. A login screen typically consists of text fields for username and password input, along with buttons for logging in and signing up. Let's build a simple login screen in SwiftUI.

Step 1: Design the User Interface

First, design the login screen interface. It will include text fields for username and password, along with login and signup buttons:

import SwiftUI
    
    struct LoginView: View {
        @State private var username = ""
        @State private var password = ""
    
        var body: some View {
            VStack {
                TextField("Username", text: $username)
                    .padding()
                    .background(Color.gray.opacity(0.1))
                    .cornerRadius(5)
                    .padding(.horizontal)
    
                SecureField("Password", text: $password)
                    .padding()
                    .background(Color.gray.opacity(0.1))
                    .cornerRadius(5)
                    .padding(.horizontal)
    
                Button(action: {
                    // Action for login button
                }) {
                    Text("Log In")
                        .font(.headline)
                        .foregroundColor(.white)
                        .padding()
                        .frame(maxWidth: .infinity)
                        .background(Color.blue)
                        .cornerRadius(5)
                        .padding(.horizontal)
                }
    
                Button(action: {
                    // Action for signup button
                }) {
                    Text("Sign Up")
                        .font(.headline)
                        .foregroundColor(.blue)
                        .padding()
                        .frame(maxWidth: .infinity)
                        .background(Color.white)
                        .cornerRadius(5)
                        .padding(.horizontal)
                        .overlay(
                            RoundedRectangle(cornerRadius: 5)
                                .stroke(Color.blue, lineWidth: 1)
                        )
                }
            }
            .padding()
        }
    }

In this example, we've created text fields for username and password input, along with login and signup buttons. The `@State` properties are used to store the input values.

Step 3: Display Error Messages

Optionally, you can display error messages to the user if login fails:

import SwiftUI
    
    struct LoginView: View {
        @State private var username = ""
        @State private var password = ""
        @State private var errorMessage = ""
    
        var body: some View {
            // UI code from Step 1...
    
                Button(action: {
                    // Authentication logic for login button
                    if username == "admin" && password == "password" {
                        // Successful login
                        errorMessage = ""
                    } else {
                        // Show error message
                        errorMessage = "Invalid username or password"
                    }
                }) {
                    // Login button UI
                }
                .alert(isPresented: $errorMessage) {
                    Alert(title: Text("Error"), message: Text(errorMessage), dismissButton: .default(Text("OK")))
                }
    
                Button(action: {
                    // Authentication logic for signup button
                    // Implement signup functionality
                }) {
                    // Signup button UI
                }
            // End of UI code from Step 1...
        }
    }

In this example, we've added an error message state variable and display it using an alert if the login fails.
